### Summary
# **Porter, M., Strang, R., & Costello, D. (2024). RTRT: Grants & Games Resource List. *Google Docs*. https://docs.google.com/document/d/1-J_cZxnau_hTxUoyxJZyXtCz3W0_OmAbCgRbNwNjWLE/edit?usp=sharing&usp=embed_facebook**

This item is a one-page Round Table Round Table resource sheet on grants and gaming, associated with Molly Porter, Rebecca Strang, and Danielle Costello. It appears to accompany a talk rather than function as an article, research study, program report, or practitioner essay. The PDF lists files for the session, grants to explore, grant-writing resources, GameRT conduct rules for Twitch, and GameRT contact/resource links.

The document's gaming-specific content points to Molly Porter's zombie game files for ALA's Thinking Money for Kids grant and to grant opportunities such as the Games & Gaming Round Table Game On! Grant, the ALA Libraries Transforming Communities grant, and Stephanie Gerding's library grants resources. The rest of the sheet points to general grant-writing support: grant resource lists, School Library Journal grant-writing tips, WebJunction proposal guidance, WebJunction grantseeking webinars, Programming Librarians' Grant Writing 101, the ALA Public Programs Office's writing grant applications resource, UConn and Adelphi grant-support guides, and books by Pamela H. MacKellar and Stephanie K. Gerding.

For the larger GBLS project, the sheet is indirectly relevant because it shows that library gaming practice depends on funding infrastructure as well as program ideas. Game collections, consoles, tabletop materials, VR gear, software access, event supplies, staff training, and inclusive community programs often require grants or external support, and professional organizations such as GameRT curate paths into those funding streams. The included Twitch policy also repeats a useful online-programming concern: gaming spaces need norms for kindness, respect, anti-harassment, moderation, and private reporting.

However, the PDF does not provide substantive evidence for the literature review. It does not describe a grant-funded library gaming program in enough detail to analyze program design, patron population, budget, staffing, game selection, outcomes, or assessment. It does not compare grant models, explain proposal strategy in a gaming-specific way, or document how libraries used grant funds to build game-based services. Most of its content is a list of links to other resources whose details would need to be evaluated directly.

This item should therefore be retained as explicitly excluded literature/resource material. It may be useful later as a starting point for a grant appendix, implementation toolkit, or project-planning resource list, but it should not be added to `review.md`. The current review needs sources that advance claims about GBLS service models, collections, instruction, community outcomes, metadata, preservation, or assessment; this resource sheet mainly supports backstage funding discovery.
